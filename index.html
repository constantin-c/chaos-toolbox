Chaos toolbox / Nude html

A big but simple bulk of all html tags you usually use in website.
The goal here is to allow you to verify your styles, and also the overall rendering & harmony.
It can also helps to prevent bug and styles conflicts (espceially on updates, it'll help prevent regressions).
And finally this will provide a solid base to create html, providing best practice, optimisations, compatibility and accessibility.

Recommandations
  • SEO / Have a proper structure, it might be used by some robots for stuff
  • SEO / Avoid using more than one H1 per page
  • SEO / Avoid skipping heading levels: start from h1, then use h2, then h3, etc.
  • SEO / Prefer using links tags inside title, and not the opposite : <h1> <a href="#"> Hey </a> </h1>
